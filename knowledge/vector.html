C++ STL 核心容器交互式教程

欢迎来到 C++ 标准模板库 (STL) 的学习之旅！本教程将通过交互式示例，带你一步步掌握 `vector`、`map` 和 `set` 这三个最常用、最强大的容器。

1. `vector` - 动态数组

1.1 定义

可以把 `vector` 想象成一个“智能”的数组。它会自动管理内存，你可以在运行时轻松地添加或删除元素。当你不知道需要存储多少元素时，它非常有用。

  • 头文件: `#include `
  • 核心特性:
    • 元素在内存中是连续存储的。
    • 可以通过索引快速访问任意元素。
    • 可以在末尾快速地添加或删除元素。

1.2 如何声明和初始化

#include <iostream>
#include <vector>
#include <string>

int main() {
    // 声明一个空的 vector，用于存储整数
    std::vector<int> scores;

    // 声明并初始化一个 vector
    std::vector<std::string> names = {"Alice", "Bob", "Charlie"};

    // 声明一个包含 5 个整数的 vector，所有元素默认初始化为 0
    std::vector<int> data(5);

    // 声明一个包含 3 个整数的 vector，并将所有元素初始化为 100
    std::vector<int> values(3, 100);
    
    return 0;
}

1.3 常规用法与操作

让我们来看一些 `vector` 的常用操作。点击“运行模拟”查看代码输出。

#include <iostream>
#include <vector>
#include <string>

void print_vector(const std::vector<std::string>& vec) {
    std::cout << "当前 Vector 中的名字: ";
    for (const std::string& name : vec) {
        std::cout << name << " ";
    }
    std::cout << std::endl;
}

int main() {
    std::vector<std::string> names;
    std::cout << "--- 添加元素 ---" << std::endl;
    names.push_back("David");
    names.push_back("Eve");
    print_vector(names);

    std::cout << "\\n--- 获取大小 ---" << std::endl;
    std::cout << "Vector 中有 " << names.size() << " 个名字。" << std::endl;

    std::cout << "\\n--- 访问元素 ---" << std::endl;
    std::cout << "第一个名字是: " << names[0] << std::endl;
    std::cout << "第二个名字是: " << names.at(1) << std::endl;

    std::cout << "\\n--- 删除末尾元素 ---" << std::endl;
    names.pop_back();
    print_vector(names);

    std::cout << "\\n--- 清空 Vector ---" << std::endl;
    names.clear();
    std::cout << "清空后的大小: " << names.size() << std::endl;
    if (names.empty()) {
        std::cout << "Vector 现在为空。" << std::endl;
    }
    return 0;
}

1.4 动手试试！

  1. 创建一个 `vector<double>` 来存储商品价格。
  2. 向其中添加几个价格，例如 `9.99`, `15.49`, `3.14`。
  3. 遍历这个 `vector` 并计算总价。
#include <iostream>

#include <numeric> // For std::accumulate

2. `map` - 键值对集合

2.1 定义

可以把 `map` 想象成一本字典。你可以通过一个唯一的“单词”（键）快速查找到它的“释义”（值）。

  • 头文件: `#include `
  • 核心特性:
    • 存储键值对 (key-value pair)。
    • 键是唯一的。
    • 内部元素默认根据键自动排序。
    • 查找、插入和删除操作的效率很高。

2.3 常规用法与操作

`map` 的强大之处在于通过键进行快速操作。

#include <iostream>
#include <map>
#include <string>

int main() {
    std::map<std::string, int> student_scores;
    std::cout << "--- 插入/更新元素 ---" << std::endl;
    student_scores["Li Ming"] = 95;
    student_scores["Zhang Wei"] = 88;
    student_scores.insert(std::make_pair("Wang Fang", 92));
    for (const auto& pair : student_scores) {
        std::cout << pair.first << " 的分数是: " << pair.second << std::endl;
    }
    
    std::cout << "\\n--- 查找键 ---" << std::endl;
    if (student_scores.count("Zhang Wei")) {
        std::cout << "找到了 Zhang Wei 的记录。" << std::endl;
    }
    
    std::cout << "\\n--- 删除元素 ---" << std::endl;
    student_scores.erase("Zhang Wei");
    std::cout << "删除 Zhang Wei 后：" << std::endl;
    for (const auto& pair : student_scores) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
    return 0;
}

2.4 动手试试！

  2. 添加三位朋友的信息。

3. `set` - 唯一元素集合

3.1 定义

`set` 是一个只存储唯一元素的集合。如果你需要一个容器来保证里面的元素不重复，`set` 是最佳选择。

  • 头文件: `#include `
  • 核心特性:
    • 所有元素都是唯一的，不会出现重复值。
    • 元素在插入时会自动排序。
    • 查找、插入和删除操作的效率很高。

3.3 常规用法与操作

`set` 的主要用途是去重和快速查找。

#include <iostream>
#include <set>
#include <vector>

int main() {
    std::set<int> unique_numbers;
    std::cout << "--- 插入元素 (5 会被忽略) ---" << std::endl;
    unique_numbers.insert(10);
    unique_numbers.insert(5);
    unique_numbers.insert(20);
    unique_numbers.insert(5);
    for (int num : unique_numbers) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    std::cout << "\\n--- 使用 set 对 vector 去重排序 ---" << std::endl;
    std::vector<int> numbers = {1, 5, 2, 3, 5, 1, 4, 2};
    std::cout << "原始 Vector: ";
    for (int n : numbers) std::cout << n << " ";
    std::cout << std::endl;
    
    std::set<int> s(numbers.begin(), numbers.end());
    std::cout << "去重排序后: ";
    for (int n : s) std::cout << n << " ";
    std::cout << std::endl;
    
    return 0;
}

3.4 动手试试！

  1. 给你一个 `vector<string>`，里面包含重复的单词：`{"apple", "banana", "apple", "orange", "banana"}`。
  2. 使用 `set` 来找出其中有多少个不同的单词。
  3. 打印出这些唯一的、按字母顺序排列的单词。
#include <iostream>
#include <vector>
#include <set>
#include <string>

int main() {
    std::vector<std::string> words = {"apple", "banana", "apple", "orange", "banana"};
    std::set<std::string> unique_words(words.begin(), words.end());
    std::cout << "不同的单词有 " << unique_words.size() << " 个:" << std::endl;
    for (const std::string& word : unique_words) {
        std::cout << word << " ";
    }
    std::cout << std::endl;
    return 0;
}

总结与选择

Vector

当你需要一个动态数组，并通过索引快速访问元素时，使用它。它是最常用的序列容器。

Map

当你需要存储键值对，并通过唯一的键快速查找、插入和删除值时，使用它。就像一本字典。

Set

当你需要存储唯一的元素，并希望它们自动排序时，使用它。非常适合用于去重。
